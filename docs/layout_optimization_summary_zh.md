# FLORIS 布局优化算法对比与解析

本文档旨在总结 FLORIS 框架中所包含的风电场布局优化（Layout Optimization）核心算法。通过横向对比算法的优劣势、适用场景及部分特殊功能，帮助开发与研究人员快速完成算法选型。

---

## 1. 核心算法对比

FLORIS 在风电场布局优化方面主要提供了以下四种算法：

### 1.1. 基于 Scipy 的布局优化 (Scipy Layout Optimization)
- **底层依赖**: `scipy.optimize.minimize` (默认基于 SLSQP)
- **优化目标**: 最大化年发电量 (AEP) 或 风场总价值 (AVP)
- **风机数量**: 固定输入变量，不支持自动优化增减
- **优点**: 
  - 集成在 Python 标准科学计算库中，依赖获取简单、开箱即用。
  - 支持在优化的每个迭代中集成开启快速尾流偏航偏转（Fast wake steering）。
  - 对于边界条件规则、地形简单凸优问题，能够极为快速地获取局部最优解。
- **缺点**: 
  - 易受尾流高非线性影响而陷入局部极值，无法保证全局最优。
  - 对于边界包含硬性避让要求或形状极其复杂的情况，梯度优化的泛化表现较弱。

### 1.2. 遗传随机搜索优化 (Genetic Random Search Layout Optimization)
- **底层依赖**: 专门设计的自研模型，融合遗传算法与随机搜索算法。
- **优化目标**: 最大化 AEP 或 价值
- **风机数量**: 固定输入变量
- **优点**: 
  - 具有**极其出色的鲁棒性**，擅长处理复杂的风况及**离散不连续的复杂边界**。
  - 支持通过廉价检查优先剔除越界错误设计，有效减少昂贵的（基于尾流模型）目标函数评估。
  - 对新手非常友好，主要是通过配置距离步长（离散概率 PMF）完成全局优化调参。
  - 易于并行化运算部署，非常不容易陷入局部最优。
- **缺点**: 
  - 随机游动机制造成收敛很慢，相较于梯度方法需要更多的循环迭代与算力堆叠。

### 1.3. 网格化布局优化 (Gridded Layout Optimization)
- **底层依赖**: 自定义几何排布算法 (扫掠与试探性平移旋转)
- **优化目标**: **最大化放入地块内部的风机总数 (N)**
- **风机数量**: **唯一将被作为优化维度的参数进行动态改变**
- **优点**: 
  - **计算极速**：完全剥离了风资源与尾流计算的负担，纯从空间几何角度判断。
  - 非常适合项目最早期的装机容量盲测、初始快速拓扑网格布局的生成。
  - 拥有常规正方形网格（Square）以及空间利用度更高的六边形网格（Hexagonal）两种选择。
- **缺点**: 
  - 绝不代表 AEP/价值 最优化，不考虑气象、尾流导致的遮挡与能量损失，往往仅只能作为进一步优化的初始解。

### 1.4. 基于 PyOptSparse 的布局优化 (PyOptSparse Layout Optimization)
- **底层依赖**: Python 第三方大型工业界联合优化框架 `pyOptSparse`。
- **优化目标**: 最大化 AEP 或 价值
- **风机数量**: 固定输入变量
- **优点**: 
  - 准工业级别的处理方案，允许调用一系列成熟稳定的顶级开源/商业级约束求解器（如 SNOPT, IPOPT 等）。
  - 非常适合数百台风机的复杂拓扑与苛刻非线性约束条件。
- **缺点**: 
  - 上手门槛陡峭。需额外安装独立的非标准包依赖及配置 C/Fortran 编译器环境，部分优质求解器必须购买商业许可 (License)。

## 2. 常见功能配置解答 (FAQ)

### Q: 上述众多布局优化算法中，哪些支持把风机数量也作为一个参数来优化？
**A: 只有“网格化布局优化 (Gridded Layout Optimization)”支持。**
在其它传统的 AEP 优化算法（如 Scipy, Random Search, PyOptSparse）中，风机的总数量 $N$ 必须作为固定常量（已知条件）。而网格化优化的底层目标并不是 AEP 寻优，而是试图在一个限定地块边界内，找出能“合法塞入”最多风机（即最大化 $N$）的具体正交或六边形平移偏转位姿组合。

### Q: 我有多个独立的离散地块属于同一个风电场，网格化布局优化 (Gridded Layout Optimization) 支持多地块约束吗？
**A: 完全支持。**
FLORIS 对于边界定义拥有强大的多边形过滤支持模块。通过向网格化布局算法的 `boundaries` 入参传入嵌套的二元元组列表 `[[(x,y), ...], [(x,y), ...]]`，算法会正确使用 `Point-In-Polygon` 检测机制将所有风电平台生成于合法的分离地块中。这种设计不仅适用于网格算法，也通用于上述提及其它的具备高鲁棒性的算法（如遗传随机搜索）。
